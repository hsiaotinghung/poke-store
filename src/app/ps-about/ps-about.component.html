Angular Pokemon Shop Web Application

This is a responsive and interactive Angular application written in Typescript. You can adjust screen size for responsive view. The breakpoints are:

  • Extra small < 576px
  • Small ≥576px
  • Medium ≥768px
  • Large ≥992px
  • Extra large ≥1200px

For more developing details please checkout source code on GitHub

Browser support

  • Chrome
  • Edge
  • Firefox
  • Safari

Dependencies

  • Angular 4.0
  • lodash
  • bootstrap
  • node-sass
  • angular-2-dropdown-multiselect
